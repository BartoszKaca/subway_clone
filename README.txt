Train Surfers to gra, w której głównym zadaniem jest uniknięcie zderzenia z pociągiem.
Pociągi jeżdżą po trzech torach, po których możemy się poruszać w prawo i w lewo.
Gra pozwala także na wskakiwanie na pociągi
W grze sterujemy postacią z perspektywy pierwszej osoby, a sama gra wykonana jest w 3D

Uwaga!
Gra uruchamia się w trybie pełnoekranowym!

Wymagania:
Python 3.11 (minimum)
biblioteka Ursina
biblioteka Screeninfo

komendy, po zainstalowaniu dołączonej wersji pythona, wykonujemy w tej lokalizacji w której znajduje się plik README

python3.11 -m pip install ursina
python3.11 -m pip install screeninfo
